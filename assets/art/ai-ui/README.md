# AI UI Asset Library

This folder contains AI-generated bitmap UI assets based on the user's preferred fairy-tale town reference style.

- `source/piece-spritesheet-v1.png`: generated source sheet for merge pieces.
- `source/ui-components-spritesheet-v1.png`: generated source sheet for UI components.
- `tiles/`: cropped 256x256 piece PNGs, including normal and advanced variants.
- `components/`: cropped UI component PNGs.
- `preview/`: quick review sheets.
- `menu/source/start-menu-v1.png`: AI-generated start menu source image.
- `characters/sprites/`: transparent bear idle/walk frames, tombstone, and money bag.

These files are the bitmap art source for the next Cocos integration pass.
